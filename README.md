# poo.xyfindables.com
Repo for poo.xyfindables.com

**Dependencies**
* Node >= 6.1

Install
```
npm install gulp
```

Setup AWS publish
```
https://docs.aws.amazon.com/cli/latest/userguide/cli-config-files.html
```

First Run
```
npm i gulp-cli -g
```

Start the project
```
gulp
#or
gulp develop
```

AWS publish
```
gulp publish
```
