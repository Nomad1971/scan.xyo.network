# scan.xyo.network
Repo for scan.xyo.network

**Dependencies**
* Node >= 7.9

Install
```
npm install gulp
```

Setup AWS publish
```
https://docs.aws.amazon.com/cli/latest/userguide/cli-config-files.html
```

First Run
```
npm i gulp-cli -g
```

Start the project
```
gulp
#or
gulp develop
```

AWS publish
```
gulp publish
```
